# Graduate Consultancy

A modern, high-performance website for **Graduate Consultancy**, built with [Next.js](https://nextjs.org/).

## 🚀 Overview

This repository contains the source code for the Graduate Consultancy website. Designed to be fast, SEO-friendly, and responsive, it serves as the digital front door for the firm, showcasing services, expertise, and contact information for prospective clients.

## ✨ Features

- **Blazing Fast Performance:** Utilizing Next.js App Router with Server-Side Rendering (SSR) and Static Site Generation (SSG).
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **SEO Optimized:** Built-in meta tags, semantic HTML, and fast load times to rank higher on search engines.
- **Modern UI:** Styled with [Tailwind CSS](https://tailwindcss.com/) for a clean, professional, and customizable interface.
- **Contact Integration:** Easy-to-use contact forms for lead generation.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** TypeScript / JavaScript
- **Deployment:** [Vercel](https://vercel.com/) (Recommended)
